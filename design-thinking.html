Applying the Design Thinking Process

This page notes how my design of this website was based on the "Design Thinking Process", a five-stage model that makes sure my website makes sense and is useful/usable.

1. Empathize

The primary goal of this website to be emphasized is to fulfill all requirements and showcase my personal portfolio as well as present myself.

2. Define

Following the rubric, clear requirements to be defined would be accessibility features, presentation of self, portfolio of >= 3 projects and hobbies, as well as demonstration of design thinking and U.N. global goals.

3. Ideate

Brainstorming of overrall UI/UX design, including layouts, color schemes and website structure. Multiple versions of the website were executed as well as overhauled, heavy inspiration was finally taken from Google LLC's Material design style, as well as corresponding fonts so that the website looked simple, concise, beautiful and usable.

4. Prototype

I built initial HTML/CSS pages, focusing on accessibility and responsive design to ensure that the webpage looked good on different scales and looked "evenly distributed" and not laggy.

5. Test

I got feedback from fellow classmates on the usability and overrall quality of the website, redid multiple versions with different layouts so that the website looked good.

This process let me produce a website that both fulfills all requirements and also looks good.
